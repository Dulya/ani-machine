Declarative animation

AniMachine is a declarative animation engine. It is based on machine state to gain flexibility.

How to declare an animation ?

You can declare an animation on DOM element by adding a HTML attribute data-am or data-am-enter

How to start the sequencer ?

Add <script>am.start()<script> at the end of your DOM.

What kind of animation can you play ?

You can play Reveal animation, Animate.css, CSShake or any other CSS animations.

How to does it works ?

Animation are launched by adding and removing CSS classes at runtime in order to preserve your own CSS styling.

If the same animations are launched, it use the same CSS classes for better performance.

When can you start animation ?

When DOM element is entering or leaving the viewport.

When events occur on element with data-am.

When other events occur on an other DOM element defined by CSS selector declared on element with data-am.

Can you chained the animations ?

Yes, you can.
