Declarative animation

AniMachine is a declarative animation engine to sexy up your frontend. It is based on machine state to gain flexibility.

How to declare animation ?

You can declare animation on DOM element by adding HTML attribute data-am or data-am-enter

How to start the sequencer ?

Add scriptam.start()script at the end of your DOM.

What kind of animation can you play ?

You can play Reveal animation, Animate.css, CSShake or any other CSS animations.

When can you start animation ?

When DOM element is entering or leaving the viewport.

When events occur on element with data-am.

When other events occur on other DOM element defined by CSS selector declared on element with data-am.

Can you chained animation ?

Yes, you can.
