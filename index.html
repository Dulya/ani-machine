Declarative animation and machine state

AniMachine is a simple declarative animation framework bases on machine state.

It allows developpers to declare multiple type of animation. It is bases on HTML declaration in attribute data-am.

What kind of animation you can do ?

You can declaration reveal animation when DOM element is entering or leaving in the viewport.

You can declaration Animate.css or CSShake animations.

How you can play the animations ?

You can play animation when an event occurs on element defined by CSS selector.
